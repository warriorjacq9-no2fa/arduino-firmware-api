For use with my [HID Keyboard firmware](https://github.com/warriorjacq9-no2fa/arduino-firmware)

# Installing

 1. Go to the green "Code" dropdown in the upper right, and select "Download ZIP"
 2. In the Arduino IDE, go to Sketch->Include Library->Add .ZIP Library... and select the downloaded ZIP

Now it is installed! To use, add `#include <HIDKeyboard.h>` to the top of your sketch
